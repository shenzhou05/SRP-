## Description

Returns true if both the inputs **A** and **B** are true. This is useful for [Branching](https://github.com/Unity-Technologies/ShaderGraph/wiki/Branch-Node).

## Ports

| Name        | Direction           | Type  | Binding | Description |
|:------------ |:-------------|:-----|:---|:---|
| A      | Input | Boolean | None | First input value |
| B      | Input | Boolean | None | Second input value |
| Out | Output      |    Boolean | None | Output value |

## Shader Function

`Out = A && B`
